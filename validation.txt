유효성 검증(form validation)
 - 사용자가 입력한 데이터에 대해서 사전 검증
 - 입력값을 받는 컨트롤러 단에서 처리
 - 자주 사용하는 유효성에 대한 검사를 도와주는 프레임워크(hibernate, validator) 사용 => @ 통해 검증가능

 - @어노테이션 종류
    1) @NotBlank : @NotEmpty + 빈문자열("") 검사
    2) @NotEmpty : 공백 허용함, 길이 0인지만 검사(공백허용 => 사용자가 공백으로 문자를 채우면 검증이 불가능함.)
    3) @NotNull : Null 체크
    4) @Length(min=2, max=5) : 입력문자 길이 제한
    5) @Email : 이메일 형식이 맞는지 검사(이메일을 입력해야 검사해줌)
    6)@Min / @Max : 지정한 값보다 큰지/작은지 검사.
    7) @Pattern : 정규식을 통한 조건    
                  regex : [a-z]+ ..
